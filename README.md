# FUTURE SKILLS

Welcome to the Skill Connect project! 

## How to Use

Follow these simple steps to clone and run the project locally on your machine.

### 1. Clone the repository
Open your terminal or command prompt and run:
```bash
git clone https://github.com/its5zoo/skill-connect.git
```

### 2. Open the project folder
Navigate into the newly cloned directory:
```bash
cd skill-connect
```

### 3. Install dependencies
Install all the required packages using npm:
```bash
npm install
```

### 4. Run the development server
Start the local server to view the app in your browser:
```bash
npm run dev
```

That's it! Open the local URL provided in the terminal (usually `http://localhost:5173`) to see the website.
